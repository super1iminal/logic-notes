# Basic Logic
- Logical analysis will help to determine if the conclusion's truth follows *necessarily* from the truth of the premises
	- Logic is sometimes defined as the science of necessary inference


### Basic Symbols
##### Statement Variables
- Symbols like *p, q* and *r* are used to represent **component sentences**

##### Misc
- ... means therefore

### Statements
A statement (or proposition) is a sentence that is *true or false* but not both

### Arguments
An argument is a sequence of statements aimed at demonstrating the truth of an assertion, ending 
##### Premises
The statements leading to the conclusion are *premises*
To be confident in a conclusion, the premises must be self-proving or follow other statements known to be true
##### Conclusion
The assertion at the end of the argumentative sequence is called the *conclusion*

### Common forms
If p or q, then r
q. 
..., r.
